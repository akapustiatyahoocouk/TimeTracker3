Dialogs

Each TimeTracker3 dialog allows user to perform a single common task, such as "create a new account", "modify the description of an existing activity" or "adjust TimeTracker3 settings and preferences".

All TimeTracker3 dialogs are modal, i.e. the user who invokes the dialog must close it before proceeding any further. For convenience, all dialogs start with their corresponding sizes and positions, but can be moved and/or resized as necessary.
